# PythonMarketplace — Платформа электронной коммерции

Веб-приложение маркетплейса с возможностью просмотра товаров, управления корзиной и заказами. Построено на **Python (FastAPI)** и **React**.

## Возможности

### Для пользователей:
- Регистрация и аутентификация (JWT)
- Просмотр каталога товаров
- Поиск товаров по названию
- Корзина покупок
- Управление заказами (просмотр, отслеживание статуса)
- Личный профиль

### Для администраторов:
- Добавление/редактирование товаров
- Управление заказами (смена статусов)
- Просмотр всех пользователей
- Удаление товаров и заказов

## Быстрый старт

### Вариант 1: Docker (рекомендуется)

```bash
# Запуск всех сервисов
docker-compose up --build

# Backend: http://localhost:8000
# Frontend: http://localhost:3000
# API Docs: http://localhost:8000/docs
```

### Вариант 2: Локальный запуск

См. подробную инструкцию в [RUN_LOCAL.md](RUN_LOCAL.md)

**Кратко:**

```bash
# Backend
cd backend
python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt
uvicorn app.main:app --reload

# Frontend (в новом терминале)
cd frontend
npm install
npm run dev
```

## API Документация

После запуска backend, Swagger UI доступен по адресу:

**http://localhost:8000/docs**

### Основные endpoints:

| Метод | Endpoint | Описание |
|-------|----------|----------|
| POST | `/api/v1/auth/register` | Регистрация пользователя |
| POST | `/api/v1/auth/login` | Вход (получение JWT) |
| GET | `/api/v1/auth/me` | Данные текущего пользователя |
| GET | `/api/v1/items` | Список товаров |
| POST | `/api/v1/items` | Создать товар (admin) |
| POST | `/api/v1/orders` | Создать заказ |
| GET | `/api/v1/orders` | Список заказов |
| GET | `/api/v1/users/{id}` | Профиль пользователя |

## Архитектура

### Схема приложения

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   Frontend  │────▶│   Backend   │────▶│ PostgreSQL  │
│   (React)   │◀────│  (FastAPI)  │◀────│             │
└─────────────┘     └─────────────┘     └─────────────┘
       │                   │
       │                   ▼
       │            ┌─────────────┐
       └────────────│   JWT Auth  │
                    └─────────────┘
```

### Модули backend:

| Модуль | Описание |
|--------|----------|
| **Auth** | Регистрация, логин, JWT токены |
| **Users** | Управление профилями, картами |
| **Items** | Каталог товаров, загрузка изображений |
| **Orders** | Создание и управление заказами |

### Роли пользователей:

- **USER** — базовые права (покупки, просмотр заказов)
- **ADMIN** — расширенные права (управление товарами, всеми заказами)
